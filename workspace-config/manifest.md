### 1. Cloud Providers
* **AWS CLI** (`aws`) — Foundation infrastructure control plane.
* **Azure CLI** (`az`) — Enterprise cloud ecosystem management.
* **Google Cloud CLI** (`gcloud`) — Core data, compute, and platform analytics engine.

### 2. Service Providers
* **Terraform CLI** (`terraform`) — Declarative Infrastructure as Code (IaC) engineering.
* **Ansible** (`ansible`) — Idempotent local and remote configuration orchestration.
* **VMware CLI** — Hybrid cloud/virtualized compute endpoint management.
* **Kubernetes CLI** (`kubectl`) — Microservice and inference container container-orchestration.
* **Helm CLI** (`helm`) — Package management for Kubernetes deployments.
* **Docker CLI** (`docker`) — Local containerization runtimes and builds.
* **GitHub CLI** (`gh`) — Source control, pull requests, and secure automation gateway.

### 3. AI Providers
* **Claude Code CLI** (`claude`) — Agentic terminal loops and context-driven engineering.
* **GitHub Copilot CLI** (`gh copilot`) — Context-aware shell assistance and predictive syntax debugging.
* **Codex CLI** — Dedicated integration layer for LLM-backed code generation engines.
